# shindomeeee.github.io

## setting

```
yarn
```

## dev-server

```
yarn start
```

## download api json

```
yarn json
```

## deploy

```
yarn deploy
```

## test

```
yarn test
# coverage mode
yarn coverage
```

## created css modules typed

```
# run watch mode
yarn tcm
```
